# Blinkit_Sales_Analysis_Project
This project analyzes Blinkit sales data to uncover insights related to customer behavior, sales performance, delivery efficiency, and product trends.
The analysis covers data cleaning, transformation, SQL-based analysis, and dashboard-ready KPIs.

The goal is to demonstrate end-to-end data analytics skills — from raw data to business insights


# Tools & Technologies

- Excel – Initial data cleaning & validation
- MySQL – Data storage, transformation, and analysis
- SQL – Joins, CTEs, Aggregations
- Power BI – Dashboard & visualization (optional extension)


# Dataset Description

The dataset contains transactional data related to Blinkit orders, including:

- Order details
- Customer information
- Product & category data
- Payment and sales values


# Data Cleaning & Preparation
Key cleaning steps performed:
-Removed duplicate records
- Handled missing values in delivery dates
- Split datetime columns into:
  - Order Date
  - Order Time
  - Delivery Date
  - Delivery Time

- Standardized column data types
- Validated row counts after import
- Checked nulls, and inconsistencies

# Database Schema (Key Tables)
- customers
- delivery
- orders
- products
- rating
- transactions


# Key KPIs
The analysis is based on the following five core KPIs:

  - Total Revenue – Total sales value generated across all orders
  - Total Orders – Number of orders placed on the platform
  - Total Quantity Sold – Total units of products sold
  - Average Order Value (AOV) – Average revenue generated per order
  - Average Product Rating – Overall customer satisfaction level based on ratings

These KPIs collectively measure sales performance, order volume, customer purchasing behavior, and product satisfaction.

# Business Questions Answered
- Which product categories drive the most revenue?
- How do monthly sales trends change over time?
- Which cities and regions perform best?
- What is the delivery performance trend?
- Which customers generate repeat business?

# Insights
- High revenue concentration in top product categories
- Sales peak during specific months (seasonality)
- Delivery delays impact customer ratings
- Few cities dominate overall revenue contribution

This project demonstrates practical skills in data cleaning, SQL analysis, KPI creation, and business storytelling, making it suitable for Data Analyst / Business Analyst roles
